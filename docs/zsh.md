# Zsh

## Project switching

| Keys | Action |
|------|--------|
| `Ctrl+P` | fzf over `~/projects`, `~/obsidian`, `~/dotfiles` → cd there |

## fzf

| Keys | Action |
|------|--------|
| `Ctrl+T` | Fuzzy file finder in current dir |
| `Ctrl+R` | Fuzzy history search |
| `Alt+C` | Fuzzy cd |

## zoxide

| Command | Action |
|---------|--------|
| `z <partial>` | Jump to most frequent matching dir |
| `zi` | Interactive dir picker (fzf) |
| `cd` | Aliased to `z` |

## Aliases

| Alias | Expands to |
|-------|-----------|
| `cd` | `z` |
| `nv` / `vn` / `vm` / `vim` | `nvim` |
